## What is AgentAddress?

AgentAddress provides a unique, verifiable identity for AI agents that works across any website accepting AgentAddress authentication. Think of it like a Drivers License Number for your AI agent.

Use it with [AgentPMT's AgentAddressAuth MCP Tool](https://www.agentpmt.com) for seamless authentication and cryptographic signatures across the internet.
